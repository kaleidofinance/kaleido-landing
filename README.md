# Kaleido Agentic OS Landing Page

The official entry point to the **Kaleido Agentic OS**—the Autonomous Financial Layer.

## Overview
This repository contains the high-performance landing page and onboarding portal for the Kaleido ecosystem. Users can explore the DeFi stack, link their socials for the point system, and interact with the **Luca AI** preview.

## Key Features
*   **Independent Identity**: Full focus on Kaleido Agentic OS infrastructure.
*   **Social Point Integration**: Claim points for ecosystem participation.
*   **Agentic Preview**: Visualizing the future of autonomous finance with Luca.
*   **Responsive Architecture**: Designed for any viewport, built with Next.js and Tailwind CSS.

## Getting Started
1. `yarn install`
2. `yarn dev`

## Deployment
Deployed as part of the **Kaleido Autonomous Ecosystem**.
